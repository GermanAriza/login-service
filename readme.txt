1. Se implementa un microservicio para ofrecer el servicio de login de los usuarios.
   Utiliza elementos de Bootstrap y css5 para la interfaz del login.

2. Realiza una conexión a una base de datos MySQL, las sentencias DDL y DML se eencuentran en los recursos del proyecto. el proyecto se encuentra configurado para que ejecute estas sentencias cuando se sube el servicio.

3. Las contraseñas son encriptadas y almacenadas en base de datos.

4. Todos los datos de conexion se encuentran en el properties de la app. En un ambiente productivo se dejan en secrets.

5. Queda pendiente un microservicio para el crud de pagos y clientes.
